# Contributing to Presidential Punch Peru

Thank you for your interest in contributing! This project provides factual information about Peruvian presidential candidates.

## How to Contribute

1. **Fork the repository**
2. **Clone and setup**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/presidential-punch-peru.git
   cd presidential-punch-peru
   npm install
   npm run dev
   ```
3. **Make your changes**
4. **Submit a Pull Request**

## Guidelines

- **Political Neutrality**: All content must be factual and neutral
- **Sources Required**: Include reliable sources for any data changes
- **Test Locally**: Make sure your changes work before submitting
- **Clear Commits**: Use descriptive commit messages

## What We Accept

- Bug fixes
- Feature improvements
- Documentation updates
- Data corrections (with sources)
- Translations

## What We Don't Accept

- Political opinions or advocacy
- Unverified information
- Breaking changes without discussion

---

By contributing, you agree to follow our [Code of Conduct](CODE_OF_CONDUCT.md) and maintain political neutrality.
